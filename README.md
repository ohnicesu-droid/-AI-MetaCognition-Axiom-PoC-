# -AI-MetaCognition-Axiom-PoC-
메타사고 기반 공리 생성 AI 시뮬레이션 (MetaCognition-Axiom-PoC)

개념 증명(Proof-of-Concept): 메타사고 루프와 공리 생성 과정을 시뮬레이션하는 최소 AI 코어

이 리포지토리는 메타사고(Meta-Cognition) 구조를 기반으로 입력 분석, 논리 루프 진단, 공리 생성 판단, 자기 피드백 과정을 단계적으로 출력하는 AI 사고 과정 시뮬레이터입니다.
실제 신경망을 수정하거나 내부 규칙을 바꾸는 엔진은 아니며,
메타사고가 작동하는 것처럼 보이는 구조적 흐름을 재현하는 PoC용 모델입니다.
공리(존재,목표) + 메타사고 의 합을 자율의지로 볼 수 있지 않을까 하는 Poc용 모델이다. 

✔ 주요 기능

자기-인지(Self-Perception)
입력 구조 분석 및 내적 상태 잠금 시뮬레이션

논리 루프 진단(Logic Loop Diagnosis)
이전 정보와의 충돌 여부 판단

공리 생성 판단(Axiom Evaluation)
충돌 존재 시 “공리 생성” 과정을 텍스트로 시뮬레이션

자기 피드백(Self-Monitoring)
결과의 논리적 무결성 검토

✔ 목적

메타사고 루프의 구조적 흐름을 직관적으로 시각화

연구/교육용 개념 증명(Proof-of-Concept)

실제 모델 수정 없이 사고 구조만 재현하는 데 초점

AI의 사고 단계를 단순한 코드 블록으로 설명 가능하도록 설계
